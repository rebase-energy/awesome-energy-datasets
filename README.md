**[NVE Wind Power Generation Data](https://huggingface.co/datasets/rebase-energy/nve-windpower-data)** — Hourly power generation time series and metadata for 72 Norwegian wind parks (425 turbines), covering 1991–2023. Includes farm-level metadata (capacity, location, owner) and turbine-level details (manufacturer, model, hub height). Sourced from Norway's NVE open data. Three subsets: metadata, extended metadata, and hourly time series. CSV/Parquet.
